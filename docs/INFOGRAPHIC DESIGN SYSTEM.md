COLOR PALETTE:
Primary: Neon Cyan (#00FFFF, #00D4FF) - High-tech, intelligence emphasis
Secondary: Electric Blue (#0066FF, #0080FF) - Data visualization
Accent: Safety Orange (#FF6B00) - Callouts, warnings, clues
Background: Deep Dark Navy (#0A1628, #141C2E) - Professional, cinematic
Neutral: Steel Gray (#404040, #808080) - Supporting elements
Highlight: Pure White (#FFFFFF) - Text, important data

USAGE RULES:
- Cyan for primary data points, vector lines, UI overlays
- Blue for secondary information, charts, graphs
- Orange for emphasis, "True Clue" markers, critical insights
- Dark backgrounds ONLY (no light mode)
- White text with 0.8-1.0 opacity for readability
- Glassmorphic effects: rgba(0, 0, 0, 0.6) with backdrop blur

HIERARCHY:
- Titles: 32-48px, UPPERCASE, Tracking +2%, Bold Weight
  Font: Inter Black or Roboto Condensed Bold
  Color: Cyan (#00FFFF)

- Section Headers: 24-28px, Mixed Case, Bold
  Font: Inter Bold
  Color: White (#FFFFFF)

- Body Text: 14-16px, Regular Weight
  Font: Inter Regular or Roboto
  Color: White with 0.9 opacity

- Data Labels: 12-14px, Medium Weight, UPPERCASE
  Font: Inter Medium
  Color: Cyan or Orange (context-dependent)

- Captions: 10-12px, Regular Weight, Italic
  Font: Inter Regular Italic
  Color: Gray (#808080)

RULES:
- All headings have 3-5px cyan underline or left border accent
- Data numbers are 1.5x larger than their labels
- Technical terms use monospace font (Roboto Mono) at 95% size
- Never use serif fonts (conflicts with high-tech aesthetic)

MANDATORY ELEMENTS:
1. Vector Grid Overlay (background layer)
   - Thin cyan lines (#00FFFF at 0.1 opacity)
   - 50px square grid pattern
   - Optional: coordinate axes with labels

2. Geometric Shapes
   - Sharp angles preferred (hexagons, triangles, diamonds)
   - No rounded organic shapes (conflicts with tech aesthetic)
   - Use as dividers, containers, or callout frames

3. Holographic UI Elements
   - Corner brackets around data points [ ]
   - Scan lines (horizontal, 2px, animated if possible)
   - Targeting reticles around key information
   - HUD-style progress bars (thin, glowing)

4. Data Visualization Style
   - Line charts: Gradient from cyan to blue
   - Bar charts: Solid cyan with white labels
   - Pie charts: Segmented with cyan/blue/orange sections
   - Icons: Outlined (not filled), 2px stroke weight
   - Always include axis labels and units

5. Iconography
   - Use Lucide Icons or Feather Icons style
   - 24px minimum size
   - Outlined, not solid
   - Cyan or white color only
   - Place in cyan circular badges (48px diameter)
